# Core Beliefs

Principles and working philosophy that guide this project.

<!-- From AGENTS.md "Core beliefs" section -->

- **Docs are the source of truth.** If the code and docs disagree, fix the code.
- **Agents implement, humans steer.** Every significant decision goes through the user.
- **One task at a time.** Finish and merge before starting the next.
- **Tests are not optional.** Every implementation task has a paired test requirement.
- **Plain English first.** This project is built by {user-role}, not an SDE. Define terms before using them.

## Adding a belief

Only add new core beliefs if they're truly foundational — these guide all major decisions. Minor guidelines go elsewhere.
