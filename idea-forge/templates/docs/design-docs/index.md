## Decision Template

When documenting a decision, use:

```markdown

## Context
What was the situation? Why did we need to decide?

## Options Considered
1. {Option A} — pros/cons
2. {Option B} — pros/cons
3. {Option C} — pros/cons

## Decision
We chose {Option X} because...

## Consequences
What changes as a result?
```
