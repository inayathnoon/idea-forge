# Review Dimensions

Score the idea across 4 dimensions (0–10 each):

- **Clarity** — Is the problem/solution crisp? Any ambiguity?
- **Feasibility** — Is the MVP realistic given constraints and stack?
- **Differentiation** — Does it solve a real problem? Obvious competition?
- **Completeness** — Missing features, edge cases, user flows?

Produce a review report with scores, strengths, weaknesses, risks, improvements, verdict.
